## Resumen

Se ha construido la base del sitio web de SC Tecnología desde una landing page básica a una estructura multi-página con SEO sólido, diseño visual alineado a la identidad de marca y preparación para escalar.

---

## Qué se ha hecho

### 1. Landing page (`index.html`)
- Diseño editorial/refinado con Manrope, azul marino `#163A5F`, naranja `#F28C28`
- Hero con titular, badge de ubicación, dos CTAs, gradientes de fondo sin imágenes
- Secciones: Hero, Trusted bar (confianza), Servicios, Sobre mí, Testimonios, Contacto, Footer
- Menú responsive con hamburguesa, smooth scroll, WhatsApp flotante

### 2. SEO implementado
- Schema `LocalBusiness` completo con servicios, fundador, contacto, área de servicio
- Meta title, description, canonical, Open Graph, Twitter Cards
- `robots.txt` y `sitemap.xml` con 6 URLs
- Jerarquía H1→H2→H3 correcta, contenido textual ampliado
- Páginas de servicio individuales (de 0 a 5 URLs indexables)

### 3. Estructura multi-página
- `/` — Landing principal
- `/servicios/desarrollo-web/`
- `/servicios/mantenimiento-informatico/`
- `/servicios/soporte-tecnico/`
- `/servicios/bases-datos-aplicaciones/`
- `/servicios/seguridad-copias-seguridad/`

### 4. Rendimiento
- Logo convertido de PNG (808 KB) a WebP (26 KB) — 97% de reducción
- Imágenes decorativas (hero-bg, about-bg, footer-bg) ya no se cargan; reemplazadas por gradientes CSS
- Sin librerías externas pesadas (no jQuery, no Bootstrap)

### 5. Formulario de contacto
- Nuevos campos: nombre, email, teléfono, mensaje
- Feedback visual al enviar
- Atributos `autocomplete` para UX
